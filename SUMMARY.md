## 💡 Key Points to Remember

1. **Google Sheet must be published**: File → Share → Publish to web
2. **Images must be public**: Right-click → Share → "Anyone with the link can view"
3. **Website auto-updates**: Every 30 seconds, no manual refresh needed
4. **COUNTIF formula**: `=COUNTIF('Closing'!A:A, H2)` in qty column
5. **Barcode in column H**: Make sure barcode is in the correct column
